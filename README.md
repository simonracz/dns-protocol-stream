# Exploring the DNS Protocol by Hand
Code created during recording the [Exploring the DNS Protocol by Hand](https://youtu.be/EDJR0DNSodg) video

# Building / Running
You need a Rust compiler, e.g. via `rustup`.
```
make
./experiment
```
